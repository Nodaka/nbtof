nbtof.nbtof_generate module¶

nbtof.nbtof_generate.nbtof_generate(notebook_names: str | list[str], func_py_name: str, func_update: bool = True)¶

This function converts single or multiple notebook files into one .py file. This .py file has functions which perform the same processings with notebooks.

Parameters:

  • notebook_names (str or list) – A notebook file name (path) or list of notebook file names. Their notebook files should be written #@ marks.

  • func_py_name (str) – A python file name (path) with functions.

  • func_upload (bool, default True) – If func_upload is True and notebooks have the same name functions, duplicated functions are removed other than the latest defined function.
